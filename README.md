# Spider-Man Landing Page 🎨🕷️

A responsive, modern landing page inspired by Spider-Man - built with **HTML** and **CSS**.  
This project was created by following a tutorial and extended with custom styling and structure.

---

## 🚀 Features
- **Hero Section**: Central Spider-Man image with hover animation.
- **Navigation Bar**: Logo and interactive menu icons (Font Awesome).
- **Social Media Links**: Vertical icons with hover effects.
- **Info Cards**: Glassmorphism-style cards highlighting awards, services, and team.
- **Responsive Design**: Uses `clamp()` for text and image sizing, plus media queries for smaller screens.
- **Dynamic Styling**: Gradients, shadows, and overlays for a cinematic look.

---

## 📂 Project Structure
```
├── index.html        # Main HTML file
├── style.css         # Stylesheet
└── images/           # Image assets (logo, Spider-Man artwork)
```

---

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling
- **Google Fonts** (Montserrat)
- **Font Awesome** for icons
